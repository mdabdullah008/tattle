### In-chat Commands to customize your experience:
<table>
          <tr>
            <th>Name</th>
            <th>Description</th>
            <th>Example</th>
          </tr>
          <tr>
            <td>!help</td>
            <td>View Commands</td>
            <td>!help</td>
          </tr>
          <tr>
            <td>!setUsername</td>
            <td>Sets Username</td>
            <td>!setUsername username</td>
          </tr>
          <tr>
            <td>!invite</td>
            <td>Gives you an invite link to the current chat room</td>
            <td>!invite</td>
          </tr>
          <tr>
            <td>!users</td>
            <td>User Count</td>
            <td>!users</td>
          </tr>
          <tr>
            <td>!color</td>
            <td>Color your message</td>
            <td>!color #00ff00 This message is green!</td>
          </tr>
          <tr>
            <td>!setColor</td>
            <td>Sets Username Color</td>
            <td>!setColor crimson</td>
          </tr>
        </table>
